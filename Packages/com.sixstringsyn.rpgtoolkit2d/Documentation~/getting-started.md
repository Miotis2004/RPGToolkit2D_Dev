# Getting Started

## Install in a Blank Unity 6 Project

1. Create or open a Unity 6 project.
2. Copy `com.sixstringsyn.rpgtoolkit2d` into the project's `Packages` folder, or keep it elsewhere on disk for local package installation.
3. In Unity, open **Window > Package Manager**.
4. Select **+ > Add package from disk...**.
5. Choose `package.json` from the `com.sixstringsyn.rpgtoolkit2d` folder.
6. Confirm **RPG Toolkit 2D** appears in Package Manager.
7. Open **Tools > RPG Toolkit > Dashboard** to verify the package foundation.

## Next Steps
