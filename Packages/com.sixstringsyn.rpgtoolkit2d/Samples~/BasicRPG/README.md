# Basic RPG Sample

This sample is intentionally lightweight for the package-preparation release. Import it from Unity Package Manager to create a project-local `Samples/RPG Toolkit 2D/<version>/BasicRPG` folder that documents the recommended first setup path without adding scene assets to your project.

## After Importing

1. Open **Tools > RPG Toolkit > Dashboard**.
2. Click **Create Authoring Folder** to create `Assets/RPGToolkit2D`.
3. Create one item, one character, and one quest definition from the dashboard content cards.
4. Run the package foundation validation from the dashboard before committing project content.

Future sample packs can add complete scenes, prefabs, and authored definitions under this folder while preserving the same Package Manager sample entry.
